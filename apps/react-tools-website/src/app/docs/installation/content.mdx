# Getting Started

React Hooks DOM provides a suite of hooks and helper functions for manipulating
the DOM.

## Installation

```s
# Using yarn
yarn add @react-tools/hooks

# Using npm
npm i add @react-tools/hooks
```

---

## Using Hooks

```jsx
import { useClick } from '@react-tools/dom';

useClick(undefined, (target) => {
  if (
    (target.closest('nav#RHD-HeaderMenu') === menuRef.current &&
      target.tagName !== 'A') ||
    target.closest('button#RHD-BurgerMenu') === burgerRef.current
  )
    return;
  else setShowMenu(false);
});
```
